Every new challenge must include:

- A `README.md` file with a general overview of the challenge and deployment instructions.
- A `flag.txt` file containing the flag
- A `challenge.yaml` file with required metadata
- A `src` folder containing any files for building challenge
- A `dist` file containing any files that should be distributed to players
- A `solution` file containing `README.md` with a challenge solution and any necessary scripts or files

Example challenge.yaml is available under /templates/challenge.yaml

Challenges without the required fields will fail automated checks and will not be accepted.
